# Vagrant Boilerplate

This boilerplate sets up one vagrant box with:

- NodeJS and NPM
- PHP
- Apache (looking in /var/www)
- Mysql (u: root, p: 1234, db: dbname)i

## Setup

1. Git clone the repo
2. Run ```vagrant up```
3. SSH into the vagrant box ```vagrant ssh```
